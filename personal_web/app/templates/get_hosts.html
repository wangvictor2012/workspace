如何修改 Hosts      Hosts 介绍

修改hosts，即替换当前系统中的hosts文件，具体替换位置如下：

** Windows 系统的Hosts 文件在：c:\windows\system32\drivers\etc\目录下
** 安卓(Android)手机系统的hosts文件在：system\etc\目录下
** Mac、Linux 类系统的在：/etc/下，修改时应当注意权限问题。

在您开始之前，建议您备份当前 hosts，如果此文件内存在其他重要内容(可用记事本打开查看，#号行表示注释掉的内容)，请以追加的形式添加; 若无则可以直接替换。

Hosts 相关

Hosts 简介：

      Hosts是一个没有扩展名的系统文件，可以用记事本等工具打开，其作用就是将一些常用的网址域名与其对应的IP地址建立一个关联“数据库”，当用户在浏览器中输入一个需要登录的网址时，系统会首先自动从Hosts文件中寻找对应的IP地址，一旦找到，系统会立即打开对应网页，如果没有找到，则系统会再将网址提交DNS域名解析服务器进行IP地址的解析。


FAQ: 常见问题：

1、如何替换hosts？

      如上述介绍，不同系统的hosts地址不一样，以windows系统为例，只需要将下载的hosts替换掉C:\ windows \ system32 \ drivers \ etc\目录下的hosts即可。


2、如何刷新 DNS 缓存？

Windows: 按下 Windows+R 键，运行 cmd，在命令提示符运行命令 ipconfig /flushdns
OS X 10.10: 在［应用程序］［实用工具］［终端］运行命令 sudo discoveryutil udnsflushcaches
OS X 10.9: 在［应用程序］［实用工具］［终端］运行命令 dscacheutil -flushcache; sudo killall -HUP mDNSResponder
OS X 10.7 ~ 10.8: 在［应用程序］［实用工具］［终端］运行命令 sudo killall -HUP mDNSResponder
Linux: 在［终端］运行命令 /etc/rc.d/init.d/nscd restart
Android、iOS: 重新启动设备


3、为什么更新后任然无法访问？

      如果更改hosts后任然无法访问，您可以尝试刷新DNS缓存试试，任然无效的话，可能是当前的hosts的IP地址不够稳定，我们会尽快提供稳定的hosts供您使用。


4、我可以自己寻找新的 IP 吗？

      可以。您可以通过其他方式找到可用的ip地址，然后通过修改hosts文件，达到访问目的。C盘hosts文件可能无法直接修改，您可以复制到桌面之后，再用记事本方式打开，修改后将原来hosts覆盖掉即可。
